DOM Popup Demo

This demo is (by motivation) functional rather than beautiful -- use CSS to tart it up.

  1. A small auto-close tooltip style popup.

  2. A draggable click-to-close popup.

  3. A second draggable click-to-close popup to demonstrate overlapping popus.

  4. An image preview click-to-open modal popup.

  5. A Dropdown menu ▾.

  6. Stacked modal message boxes.

DOM Popup webpage: http://www.methods.co.nz/popup/popup.html
